关于博客的建立过程（一）

发表于 2022-02-08
这篇文章来讲一讲建立三语博客的历程。

这篇文章来讲一讲整个博客的建立过程。

首先其实一开始并没有建立博客的想法，最初只是为了搞个域名邮箱，顺手申请了域名。而当域名到手里后，却又觉得仅仅用来做邮箱似乎有些浪费，那么何不趁此顺手做一个个人博客试试呢？如你所见，便有了这个博客。

而折腾的过程总是充满着乐趣。在网上看了各种搭建博客的推荐帖后，结合自己的实际需求，最终选择了采用 Hexo + GitHub Pages 这种简单实际的静态博客：虽然稍微折腾一些，但经济实用，不需要自己再搭建服务器，而自己的博客也本身没有追求流量和收入，也就无所谓什么 SEO 之类的。

建立网站很简单，基本网上教程一搜一大把，比如我就基本参考的这篇文章。而建立后的个性化却是见仁见智的了。我的本意是建立一个中英德三语的博客，可以顺手练习自己的德语水平（但愿吧……），但似乎这个需求并没有很大，只找到了少许的教程。而这个主题的国际化似乎也并没有达到我理想的状态，最多也就只能加一个切换语言的按钮，实际的内容却在测试中总是出这样那样的错误。我一度考虑 WordPress 这种，简单搜索了一下，似乎 WordPress 的国际化也没有现成的方案，也需要靠额外的插件来实现（见），于是作罢。

后来看到了这篇文章，博主的方案倒是基本符合我的设想，只是一来需要额外在每篇文章加上切换按钮，感觉不够优雅；二来要单独在菜单栏加一个切换语言的按钮，而且这个按钮点了只能切换到另一个语言的主页，不是基于本页面切换；三来就是没有用上主题本身自带的语言切换器。于是经过一些摸索，我在原博主的基础上改进了些许，似乎看起来是达到了我基本的设想：使用主题自带的切换按钮，点击后直接切换本文的语言而不是返回主页。
